# Finance Manager Auth Template

Small Node.js + TypeScript app with Express, SQLite, bcrypt, cookies, sessions,
login, protected dashboard, and logout.

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

Default login:

```text
admin / senhaqtuvaiusar
```

## Scripts

- `npm run dev` runs the TypeScript server.
- `npm run build` compiles to `dist/`.
- `npm start` runs the compiled app.
